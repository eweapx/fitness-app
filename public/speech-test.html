Speech Recognition Test

Click the button below to test if your browser supports speech recognition and if the microphone is working properly.

Test Instructions
Recommended test phrases:
  • "Log an activity called morning run for 30 minutes"
  • "Add a meal called breakfast with 500 calories"
  • "Log 8 hours of sleep"
  • "Log 10000 steps"
Troubleshooting:
  • Make sure your microphone is enabled in browser settings
  • Try using Chrome or Edge browsers for best compatibility
  • Speak clearly and at a normal pace
  • Reload the page if speech recognition stops responding
Voice Commands Guide

All commands should start with "log" or "add" followed by the type of data:

  • Activities: "log activity", "log workout", "add exercise"
  • Nutrition: "log meal", "add food", "log nutrition"
  • Sleep: "log sleep", "add sleep"
  • Habits: "log habit", "add habit"
  • Steps: "log steps", "add steps"
